# Expense Tracker (Python CLI)

A lightweight command-line **expense tracker** written in Python.  
It helps you record expenses, set a monthly budget, and view summaries.

## Features
- Add new expenses (amount, category, date/notes)
- Set and track a monthly budget
- Summarize spending by category
- View all recorded expenses
- CSV-based storage (simple and portable)
- Colorful terminal output
- Daily allowance calculation based on remaining budget and days in the month

## Tech Stack
- Python
- CSV file storage
- CLI (Command Line Interface)

## Run
```bash
python expense_tracker.py
